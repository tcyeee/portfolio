## 摘要

MacOS相比Windows去除了很多花哨的功能,让我在工作的时候更加专注,其自带的各种全局快捷键也极大的提高了工作效率,加上其简洁的特性和流畅的应用过渡让人印象深刻,使用过一段时间以后再回去用win,那种低效率的任务切换和鸡肋的多桌面设计让我忍不住吐槽,虽然最近几次更新有在多任务界面做优化,但是习惯了使用macOS方便的手势任务切换以后,就再也回不去了...

Macos系统带有非常多且好用的手势,可以很方便炫酷的实现各种操作.

可是,我偏不爱用!

与我而言无论使用鼠标还是触控板去操作电脑,都是一件非常笨重的且低效的交互方法,于是我计划将所有的触控操作都投射到键盘快捷键上,且达到以下目的:

1. 必须单手可以完成.
2. 必须符合人体工学,不会累,而且要适配40%键盘.

于是自己设计一套更适合自己的映射

## 1.相关知识介绍

### 1.1 MacOS 修饰键映射

Mac修饰键映射是内置于macos系统内的键盘设置中,作用是将物理案件和软件中的信号接受进行关联,如果你使用的是window键盘, 则可以使用这个映射很方便的将Windos系统键盘中的 `Control`键, `WIN`键, `Alt`键映射到mac系统中的 `⌥` / `⌃` / `⌘` 键上.

MacOS中可以同时使用多个键盘, 在这里可以对每一种键盘都可以自定义布局, mini键盘和内置键盘有出入, 这里单独设置

## 2.功能的实现

**替换全部触控动作为键盘单手按键**

| shortcut key | function       | shortcut key | function   |
| ------------ | -------------- | ------------ | ---------- |
| `⌘` + `Esc`  | 应用切换(搜索) | `⌥` + `A`    | 上一个桌面 |
| `⌘` + `Tab`  | 应用切换(图标) | `⌥` + `D`    | 下一个桌面 |
| `⌘` + `Q`    | 应用切换(图片) | `⌥` + `S`    | 返回主桌面 |
| `⌘` + `E`    | 应用选择(列表) |              |            |
|              |                |              |            |
