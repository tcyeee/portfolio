# Raven IQ Test Mini Program

A lightweight and open-source WeChat Mini Program that delivers a fast, standardized Raven's Progressive Matrices intelligence test. Users can complete the assessment in under three minutes with a simple and intuitive interface.

## Features

- Standardized Raven's Progressive Matrices test
- Three-minute quick assessment mode
- Minimal and clean UI optimized for WeChat Mini Program
- Local scoring and instant results
- Fully open source for learning and customization

## Tech Stack

- WeChat Mini Program
- JavaScript / WXML / WXSS
- Lightweight logic and zero external dependencies

## Getting Started

Clone the repository:

```shell
git clone <your-repo-url>
```

Open the project with WeChat DevTools, and run the Mini Program directly.

## Project Goals

This project aims to provide a fast, accessible, and accurate cognitive assessment tool while serving as a reference for developers building Mini Programs with clean UI and simple logic.
